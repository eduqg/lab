## Getting Started

```console
yarn
yarn dev
```

## Dependencies

```console
yarn add @vercel/node
yarn add axios
yarn add mongodb
yarn add @types/mongodb -D
```

## Vercel

```console
vercel login
vercel
vercel --prod
```
