# Projeto Sass

Instalar globalmente o sass

```console
npm install -g sass
```

Ler sass e compilar em css

```console
sass style.scss:style.css
sass style.scss:style.css --watch
```

Criar arquivo reset.css

Resetar CSS Meyer Web
